  • A specific simulator or device is called a backend.

  • By default, simulators execute shots=1024 realizations of a circuit under ideal condition.

Bernstein-Vazirani#

\(f:\{0, 1\}^n \rightarrow \{0, 1\}\) where \(f(x) = x\cdot a~(mod2)\) with \(a \in \{0, 1\}^n\)

Find \(a\) where \(n = 3\).

def BV(U):
    num_str = U.num_qubits - 1
    qc = QuantumCircuit(num_str + 1, num_str)
    qc.x(num_str)
    qc.h(range(num_str + 1))
    qc.barrier()
    qc.append(U, range(num_str+1))
    qc.barrier()
    qc.h(range(num_str))
    qc.measure(range(num_str), range(num_str))
    
    return qc

def teleportation(nq): #nq : the total number of qubits in the qc, should be odd number
    q = QuantumRegister(nq, 'q')
    c = ClassicalRegister(nq-1, 'c')

    qc = QuantumCircuit(q, c)
    qc.append(U, [0])
    for k in range(1, q.size, 2):
        qc.append(bell_ent, range(k, k+2))
        qc.append(bell_meas, range(k-1, k+1))

    qc.measure(range(c.size), range(c.size))    

    for k in range(0, c.size, 2):
        qc.z(c.size).c_if(c[k], 1)
        qc.x(c.size).c_if(c[k+1], 1)

    qc.snapshot('st_all')
    return qc

Hands-on exercise 3.

Let's build a quantum circuit for a protocol called 'one-bit teleportation' which does not consume an entangled state.

Suppose that Alice and Bob are close together, such that they can perform a CNOT gate on their qubits. Alice begins with the state \(|\psi\rangle = \alpha|0\rangle + \beta|1\rangle\), where the value \(\alpha\) and \(\beta\) are up to you. Bob begins with the state \(|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)\). We want the state of Bobs qubit is \(|\psi\rangle\) at the end of the protocol.

Construct a quantum circuit composed of single qubit gates to prepare Alice and Bobs qubits initial states accordingly, and CNOT gate that has right direction for this protocol. Finally measure Alices qubit and perform an appropriate gate on Bobs qubit based on the measurement result.

To implement the conditional gate in Qiskit, you need to apply the method c_if that was used in the teleportation circuit above.

import numpy.linalg as la

## your code goes here

my_telep = QuantumCircuit(2)

# # Initialize 
# my_telep.append(random_unitary(2), [0]) # for Alice's qubit
# my_telep.h(1) # for Bob's qubit

# Initialize (alternative way)
vec = np.random.rand(4)
st_alice = [vec[0]+1j*vec[1], vec[2]+1j*vec[3]]
st_alice /= la.norm(st_alice)
my_telep.initialize(st_alice, 0)
my_telep.h(1)

# Obtain the density matrix for q0 (to check the teleportation result later)
dm0 = DensityMatrix(my_telep)
dm_orig_alice = partial_trace(dm0, [1])

# CNOT operation
my_telep.cx(1, 0)

# measure q0 (Alice's qubit) and apply a gate on q1 (Bob's qubit) 
# conditioned on the meausrement value.

c = ClassicalRegister(1, 'c')
my_telep.add_register(c)
my_telep.measure(0, 0)
my_telep.x(1).c_if(c[0], 1)

my_telep.draw('mpl')

# simulate the circuit 
my_telep.snapshot('st_all')
my_telep_trans = transpile(my_telep, backend)
result = backend.run(my_telep).result()
data_all = result.data()['snapshots']['statevector']
dm1 = DensityMatrix(data_all['st_all'][0])
dm_telep_bob = partial_trace(dm1, [0])

# check density matrix for q1
print(dm_orig_alice == dm_telep_bob)
