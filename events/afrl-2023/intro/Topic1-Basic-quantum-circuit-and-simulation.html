Ex 1. Bernstein-Vazirani#

  • Qubit ordering

\(f:\{0, 1\}^n \rightarrow \{0, 1\}\) where \(f(x) = x\cdot a~(mod2)\) with \(a \in \{0, 1\}^n\)

Find \(a\).

a = '1101'
nq = len(a)+1

bv = QuantumCircuit(nq, nq-1)
bv.x(nq-1)
bv.h(range(nq))
bv.barrier()
bv.cx([0, 2, 3], nq-1)
bv.barrier()
bv.h(range(nq))
bv.barrier()
bv.measure(range(nq-1), range(nq-1))

Ex 3. Quantum Phase Estimation#

Simple QPE circuit with a phase gate#

  • Circuit library

(1)#\[\begin{equation} U = \begin{pmatrix} 1 & 0 \\ 0 & e^{i\frac{\pi}{4}}\\ \end{pmatrix} \end{equation}\]
from qiskit.circuit.library import QFT

# simple qpe circuit with a phase gate
phase0 = np.pi/4
n_prec0 = 3

qpe_simple = QuantumCircuit(n_prec0+1, n_prec0)
qpe_simple.h(range(n_prec0))
qpe_simple.x(n_prec0)
for k in range(n_prec0):
    qpe_simple.cp(phase0*2**k, k, n_prec0)
qpe_simple.append(QFT(n_prec0).inverse(), range(n_prec0))
qpe_simple.measure(range(n_prec0), range(n_prec0))

qpe_simple.draw('mpl')

  • ‘initialize’ method

  • Build a custom gate

from qiskit.quantum_info import random_unitary
import numpy.linalg as la

quantum_info module

# Phase to estimate
n_st = 2
Urand = random_unitary(2**n_st)
vals, vecs = la.eig(Urand)
val, st = vals[1], vecs[:, 1]
if np.angle(val) < 0:
    phase = 2*np.pi + np.angle(val)
else:
    phase = np.angle(val)
    
print('phase to estimate: {}'.format(np.round(phase, 3)))

phase to estimate: 2.704

# circuit for eigenstate prep
qc_st = QuantumCircuit(n_st)
qc_st.initialize(st)
qc_st.draw('mpl')

n_prec = 6
U = Urand.to_instruction()

qpe = QuantumCircuit(n_prec+n_st, n_prec)
qpe.h(range(n_prec))
qpe.append(qc_st, [-2, -1])

for k in range(n_prec):
    qpe.append(U.power(2**k).control(), [k, -2, -1])

qpe.append(QFT(n_prec).inverse(), range(n_prec))
qpe.measure(range(n_prec), range(n_prec))
qpe.draw('mpl')

from qiskit_aer import AerSimulator

AerSimulator#

  • qiskit_aer is a qiskit simulation package.

  • AerSimulator simulates experiments and returns counts.

  • By default, AerSimulator execute shots=1024 realizations of a circuit under ideal condition.

  • A specific simulator or device is called a backend.

  • backend.run takes a quantum circuit, or list of circuits, and execute the circuits on the chosen backend.

backend = AerSimulator()
