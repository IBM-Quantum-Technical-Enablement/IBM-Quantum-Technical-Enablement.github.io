Topic 4. Noisy Simulation#

Instructor : Hwajung Kang, Ph.D., Education Delivery Program Manager, IBM Quantum

from qiskit import *
from qiskit.visualization import plot_distribution, plot_error_map
from qiskit.providers.fake_provider import FakeHanoi

from qiskit_ibm_provider import IBMProvider

from qiskit_aer import AerSimulator
from qiskit_aer.noise import (NoiseModel, ReadoutError,
                              depolarizing_error, thermal_relaxation_error)

import numpy as np
import matplotlib.pyplot as plt

a = '1111'
nq = len(a)+1

bv = QuantumCircuit(nq, nq-1)
bv.x(nq-1)
bv.h(range(nq))
bv.barrier()
bv.cx(range(len(a)), nq-1)
bv.barrier()
bv.h(range(nq))
bv.barrier()
bv.measure(range(nq-1), range(nq-1))

bv.draw('mpl')

Steps

  1. Use the appropriate classes or functions from Qiskit Aer noise module to construct the errors that you want.

  2. Build a noise model by adding the errors to an instance of the NoiseModel class.

  3. Pass the noise model to the AerSimulator.

  4. Perform noisy simulation.

Readout error#

ghz = QuantumCircuit(5)
ghz.h(0)
ghz.cx(0, range(1, 5))
ghz.measure_all()
ghz.draw('mpl')

  • Parameters : a list of assignment probability vectors \(P(A|B)\) where \(A\) and \(B\) are noisy measurement outcomes and true measurement oucomes, respectively.
    Explicitly, \(P(A|B)= [P(A|k)]\) where \(k\in\{0, 1, 2, ..., 2^n - 1\}\) for \(n\) number of qubits.

    E.g. for 1 qubit case, \(P(A|B) = [P(A|0), P(A|1)] = [[P(0|0), P(1|0)], [P(0|1), P(1|1)]]\).

p10, p01 = 0.01, 0.03
ro = ReadoutError([[1-p10, p10],[p01, 1-p01]])

meas_noise = NoiseModel()
meas_noise.add_all_qubit_readout_error(ro)
#meas_noise.add_readout_error(ro,[0])

noiseSim = AerSimulator(noise_model=meas_noise)

ghz_trans = transpile(ghz, noiseSim)
counts_ghz = noiseSim.run(ghz_trans, shots=100000).result().get_counts()

plot_distribution(counts_ghz)

Quantum errors#

Applying both depolarizing and thermal relaxation errors on the cx gates

#only on cx
p = 5e-2 
t1, t2, t_cx = 100e3, 50e3, 600 # in nano second 

depol_cx = depolarizing_error(p, 2)
thermal_cx = thermal_relaxation_error(t1, t2, t_cx)
err_cx = depol_cx.compose(thermal_cx)

print(err_cx)
