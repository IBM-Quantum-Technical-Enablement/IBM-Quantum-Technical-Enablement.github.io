Custom Noisy Simulation#

Build Readout Noise model using ReadoutError class#

Check ReadoutError

  • Parameters : a list of assignment probability vectors \(P(A|B)\) where \(A\) and \(B\) are noisy measurement outcomes and true measurement oucomes, respectively.
    Explicitly, \(P(A|B)= [P(A|k)]\) where \(k\in\{0, 1, 2, ..., 2^n - 1\}\) for \(n\) number of qubits.

    E.g. for 1 qubit case, \(P(A|B) = [P(A|0), P(A|1)] = [[P(0|0), P(1|0)], [P(0|1), P(1|1)]]\).

p10, p01 = 0.1, 0.2
ro = ReadoutError([[1-p10, p10], [p01, 1-p01]])
print(ro)

ro_noise = NoiseModel()

ro_noise.add_all_qubit_readout_error(ro)
## alternative
# ro_noise.add_readout_error(ro, [0])
# ro_noise.add_readout_error(ro, [1])
# ro_noise.add_readout_error(ro, [2])
# ro_noise.add_readout_error(ro, [3])

ReadoutError on 1 qubits. Assignment probabilities:
 P(j|0) =  [0.9 0.1]
 P(j|1) =  [0.2 0.8]
